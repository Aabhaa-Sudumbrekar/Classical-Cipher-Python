# Classical Cipher Implementations (Python)

This repository contains Python implementations of classical encryption techniques. Each program demonstrates how traditional ciphers work by converting text using basic mathematical transformations.

## 🔐 Current Project: Caesar Cipher

A simple implementation of the Caesar cipher that supports:

- Encryption of plaintext
- Decryption of ciphertext
- Custom shift key
- Preservation of spaces and special characters
- Uppercase text handling
